Це настільки не зв'язане з бізнес-логікою завдання, що для цього навіть не потрібно писати код. 
Ви будете використовувати готові програми, такі як nginx або apache, щоб сервити файли, аналізувати заголовки Host хедерів та різних edge cases, і не робити всю цю рутину власноруч. 
Це називається "хостинг статичних файлів" (в тому сенсі, що бекенд буде просто надавати вміст файлів, бездумно) або просто "сервити статику". І ще, статику можна розміщувати на Github Pages або Cloudflare Pages.
